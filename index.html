Python & RenPy

SCRIPTING / VISUAL NOVELS

Python es el lenguaje base: scripts, automatizaciones, bots, herramientas. Versátil, legible y potente. Lo uso para todo lo que necesita lógica rápida sin overhead.

RenPy extiende Python para crear visual novels con sistema de diálogos, ramas narrativas, manejo de sprites, música y variables de historia. Ideal para proyectos narrativos con profundidad.

Python 3.x RenPy SDK Visual Novels Scripting Automatización

Clickteam Fusion

EVENT-DRIVEN GAME DEV

Clickteam Fusion 2.5 es un entorno de desarrollo de juegos basado en eventos. Sin escribir código tradicional, se construyen lógicas complejas mediante condiciones y acciones visuales.

Muy usado para juegos 2D indie clásicos. Su exportador permite builds para Windows, HTML5 y móvil. Perfecto para prototipos rápidos y juegos de plataformas o puzzles.

Event Editor 2D Games Indie Dev Windows / HTML5

GameMaker

2D GAME ENGINE / GML

GameMaker Studio es el motor detrás de grandes títulos indie como Undertale, Hotline Miami o Spelunky. GML (GameMaker Language) es específico del engine pero extremadamente eficiente para 2D.

Su flujo de trabajo combina editor visual con scripting directo. Ideal para juegos 2D pixel art, roguelikes, metroidvanias y cualquier proyecto que priorice el rendimiento en 2D.

GML Pixel Art 2D Expert Multiplatform

Unity

3D / 2D ENGINE / C#

Unity es el motor más completo del catálogo. Programado en C#, permite crear experiencias en 2D y 3D con un ecosistema enorme de assets, paquetes y documentación.

Su editor es potente: sistema de física, animación, shaders, UI, networking. Exporta a más de 25 plataformas incluyendo PC, consolas, móvil y WebGL. El estándar de la industria indie y AA.

C# 3D / 2D HLSL Shaders 25+ Plataformas Asset Store
