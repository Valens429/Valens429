Python & RenPy

SCRIPTING / VISUAL NOVELS

Python es el lenguaje base: scripts, automatizaciones, bots, herramientas. Versátil, legible y potente.

RenPy extiende Python para crear visual novels con sistema de diálogos, ramas narrativas, manejo de sprites, música y variables de historia.

Python 3.xRenPy SDKVisual NovelsScriptingAutomatización

Clickteam Fusion

EVENT-DRIVEN GAME DEV

Clickteam Fusion 2.5 es un entorno de desarrollo de juegos basado en eventos. Sin escribir código tradicional, se construyen lógicas complejas mediante condiciones y acciones visuales.

Muy usado para juegos 2D indie clásicos. Exporta a Windows, HTML5 y móvil.

Event Editor2D GamesIndie DevWindows / HTML5

GameMaker

2D GAME ENGINE / GML

GameMaker Studio es el motor detrás de grandes títulos indie como Undertale, Hotline Miami o Spelunky. GML es específico del engine pero extremadamente eficiente para 2D.

Su flujo de trabajo combina editor visual con scripting directo. Ideal para juegos 2D pixel art, roguelikes y metroidvanias.

GMLPixel Art2D ExpertMultiplatform

Unity

3D / 2D ENGINE / C#

Unity es el motor más completo del catálogo. Programado en C#, permite crear experiencias en 2D y 3D con un ecosistema enorme de assets, paquetes y documentación.

Su editor incluye sistema de física, animación, shaders, UI y networking. Exporta a más de 25 plataformas.

C#3D / 2DHLSL Shaders25+ PlataformasAsset Store
